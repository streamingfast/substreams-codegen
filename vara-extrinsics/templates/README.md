# Vara Extrinsic

This Substreams project allows you to retrieve Vara transactions filtered by one or several Extrincis (i.e. you will only receive transactions containing the specified Extrinsics).

## Get Started

### Build the Substreams

```bash
substreams build
```

### Authenticate

```bash
substreams auth
```

### Run your Substreams

```bash
substreams gui
```
